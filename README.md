> 인프런. 김왼손의 왼손코딩(한입에 쏙 파이썬)

## 파이썬의 장점

- 다른 언어에 비해 쉽게 배울 수 있다.
- 참고할 자료가 많다.
- 빠르게 만들 수 있다.

## 파이썬으로 할 수 있는 것

- 구글의 핵심 3대 언어 중 하나
- 나사, 인스타그램등 많은 곳에서 사용 중
- 데이터 과학
- 인공지능
- 자동화 프로그램
- 게임(장고, 플라스크 사용해서)
- 웹 개발
